Dentro do projeto tem q mudar o arquivo '.env' para conectar com seu MYSQL, isso deve ser feito colocando os seus dados aqui:
DB_CONNECTION=mysql
DB_HOST=Host
DB_PORT=Porta
DB_DATABASE=Nome do banco
DB_USERNAME=Nome do usuário
DB_PASSWORD=Senha

Exemplo de preenchimento:
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3307
DB_DATABASE=bdtcc
DB_USERNAME=root
DB_PASSWORD=12345
